## Overview

시네씨드는 사용자의 영화 취향을 기반으로 독립영화를 추천하고,  
제작 과정에 대한 정보를 제공하는 플랫폼 서비스입니다.

소수 인원 프로젝트로, 기획 의도와 구현 디테일을 빠르게 맞춰가는 데 집중했습니다.

## My Role

- UX/UI 디자인 총괄
- ERD 설계
- 영화 취향 토너먼트 게임 개발
- 영화 추천 기능 구현

## Key Features & Implementation

- 사용자의 선택에 따라 추천 결과가 달라지는 영화 취향 토너먼트 게임 구현
- 토너먼트 단계별 상태와 결과 계산 로직 분리
- 영화 추천 결과 화면 UI 구성
- ERD 설계를 통해 데이터 구조를 명확히 정의하고, 프론트엔드–백엔드 간 협업 효율 개선

## Design Consideration

- 선택 과정이 복잡하지 않도록 단계별 화면 흐름을 단순화
- 추천 결과에 대한 신뢰감을 주기 위해 시각적 강조 요소 활용

## What I Learned

- 소규모 팀에서의 빠른 의사결정과 역할 분담의 중요성
- 게임형 인터랙션에서 상태 관리와 로직 분리의 필요성
