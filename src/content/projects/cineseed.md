## Overview

시네씨드는 사용자의 영화 취향을 기반으로 독립영화를 추천하고,  
제작 과정에 대한 정보를 제공하는 플랫폼 서비스입니다.

소수 인원 프로젝트로, 기획 의도와 구현 디테일을 빠르게 맞춰가는 데 집중했습니다.

## Key Features

- 독립 영화 펀딩 및 제작 참여 신청 기능
- 영화 취향을 탐색할 수 있는 토너먼트형 추천 인터랙션
- 사용자 선택 결과에 따른 맞춤형 영화 추천 화면 제공

## What I Did

- 사용자의 선택에 따라 추천 결과가 달라지는 영화 취향 토너먼트 게임 로직 설계 및 구현
- 토너먼트 단계별 상태, 사용자 선택, 최종 결과 계산 로직을 분리해 관리
- 추천 결과 화면 UI 구성 및 사용자 흐름 정리
- ERD 설계를 통해 데이터 구조를 정의하고, 프론트엔드–백엔드 간 데이터 흐름을 명확히 정리

## What I Learned

- 소규모 팀에서의 빠른 의사결정과 역할 분담의 중요성
- 게임형 인터랙션에서 상태 관리와 로직 분리의 필요성
