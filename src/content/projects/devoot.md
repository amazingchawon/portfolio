## Overview

개발바닥은 다양한 IT 강의를 비교하고, 학습 과정을 기록할 수 있도록 돕는 개발자 전용 SNS 서비스입니다.  
학습 이력 관리와 커뮤니티 기능을 결합해, 개발자의 지속적인 성장을 지원하는 것을 목표로 했습니다.

## My Role

- UX/UI 디자인 총괄
- PM 역할로 일정 관리 및 태스크 분배
- 웹 프론트엔드 개발
- 관리자 페이지 개발

기획부터 디자인, 개발까지 전반적인 흐름을 조율하며 프로젝트를 리드했습니다.

## Key Features & Implementation

- Firebase OAuth를 활용한 회원 가입 및 로그인 기능 구현
- 마이 페이지 및 강의 상세 페이지 개발
- 반응형 네비게이션 바 제작
- Pinia 기반 로그인 상태 및 전역 상태 관리 설계
- 댓글 기능 구현 및 상태 갱신 처리
- 관리자 페이지를 통해 강의 및 사용자 관리 기능 구현
- Figma로 설계한 컴포넌트를 재사용 가능한 구조로 프론트엔드에 반영

## Process & Collaboration

- PM으로서 Jira를 활용해 전체 일정 관리 및 역할 분담
- 기획–디자인–개발 간 흐름이 끊기지 않도록 커뮤니케이션 조율

## What I Learned

- 북마크 기능에서 상태 변경 시마다 API를 호출하지 않고,  
  최종 변경 시점에만 요청하도록 개선하여 리소스 효율을 높인 경험
- 다양한 역할을 동시에 수행하며 프로젝트 전반을 통합적으로 바라보는 시야를 확장
