## Overview

볼로그는 취미 풋살인들을 대상으로, 별도의 장비 없이 스마트워치만을 사용해 경기력을 관리할 수 있도록 도와주는 Android 모바일 서비스입니다.
경기 활동을 한눈에 이해할 수 있도록 히트맵과 캘린더 중심의 UI를 설계했습니다.

## Key Features

- Samsung Health SDK를 활용한 경기 데이터 수집
- GPS·속도 데이터를 가공해 매치 리포트 제공

## What I Did

- 클라이언트–서버 통신 구조 설계 참여
- 사용자 테스트를 주도하며 UX 개선 반복
- GPS 데이터 기반 히트맵 시각화
- Jetpack Compose 기반 캘린더 UI 구현
- 사용자 피드백을 바탕으로 경기 등록 플로우 UX 개선

## Validation & Iteration

실제 풋살 팀을 대상으로 테스트를 진행하고,  
피드백을 바탕으로 경기 등록 플로우와 입력 UX를 개선했습니다.

## What I Learned

- Jetpack Compose를 활용한 안드로이드 UI 개발 경험
- MVP 기반 개발과 사용자 피드백을 통한 반복 개선 과정의 가치
