## Overview

볼로그는 스마트워치로 수집한 활동 데이터를 바탕으로 풋살 경기 기록과 흐름을 시각화하는 Android 모바일 서비스입니다.  
히트맵, 경기 기록, 캘린더 기능을 중심으로 사용자의 경기 활동을 한눈에 파악할 수 있도록 설계했습니다.

## My Role

- UX/UI 디자인 총괄
- REST API 설계
- 안드로이드 프론트엔드 개발 (3인 협업)

디자인 전공자로서 사용자 흐름과 정보 밀도를 우선 고려해 화면 구조를 설계했고, 개발 단계에서는 실제 구현 가능성과 상태 관리 구조를 함께 고민했습니다.

## Key Features & Implementation

- 사용자 흐름 기반 UI/UX 설계 및 Figma 프로토타입 제작
- REST API 설계에 참여하여 클라이언트 요청 구조 정의
- 히트맵 UI 컴포넌트 구현을 통해 이동 경로 시각화
- 경기 데이터 화면 구성 및 인터랙션 처리
- Jetpack Compose 기반 캘린더 UI 구현
- 경기 등록 기능 개발

## Validation & Iteration

실제 풋살 팀을 대상으로 테스트를 진행하고,  
피드백을 바탕으로 경기 등록 플로우와 입력 UX를 개선했습니다.

## What I Learned

- Jetpack Compose를 활용한 안드로이드 UI 개발 경험
- 디자이너–개발자 간 원활한 커뮤니케이션의 중요성
- MVP 기반 개발과 사용자 피드백을 통한 반복 개선 과정의 가치
