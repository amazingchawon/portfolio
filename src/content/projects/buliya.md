## Overview

지하 상가 화재 상황에서 사용자가 현재 상황을 빠르게 파악하기 어렵다는 문제에서 출발했습니다.  
실시간 CCTV와 지도 정보를 기반으로 비상 상황을 시각화하는 화면 흐름과 관리자용 웹 UI를 구현했습니다.

## What I Did

- 지도 기반 화면 흐름 설계 및 UI 구현
- 상태 변화에 따른 시각적 강조(경고/우선순위) 처리
- 공통 UI 컴포넌트 정리 및 스타일 일관성 유지

## Key Points

- 지도에서 상황 → 대피 정보로 이어지는 정보 구조를 단순화
- 사용자의 판단 시간을 줄이기 위한 강조 규칙 설계
